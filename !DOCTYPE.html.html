Driver Drowsiness Detection AI

Powered by Teachable Machine & TensorFlow.js

What is this?

This is a real-time AI-powered system that detects whether a driver is awake or drowsy using webcam input. It uses machine learning models trained with Teachable Machine to classify driver alertness.

Why is it important?

Driver fatigue is one of the major causes of road accidents. This tool can help prevent such accidents by alerting drivers with a voice warning when signs of drowsiness are detected.

How does it work?

The model continuously monitors the driver's face using a webcam. If the system detects that the user is drowsy for more than 3 seconds, it plays an audio alert to wake them up. All processing happens locally in the browser â€” no data is stored or uploaded.
